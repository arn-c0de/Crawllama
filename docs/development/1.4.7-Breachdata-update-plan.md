│ Plan to implement                                                           │
│                                                                             │
│ Modular Breach Detection Architecture                                       │
│                                                                             │
│ Context                                                                     │
│                                                                             │
│ email_intel.py (1417 lines) is monolithic -- all breach detection logic for │
│  HIBP, LeakCheck, IntelX, DeHashed, local DB scanning, and placeholder      │
│ methods live as private methods across two classes (EmailIntelligence and   │
│ EmailVulnerabilityIntel). Adding new sources means editing this file, and   │
│ each source handles its own HTTP, rate limiting, and error handling         │
│ inconsistently.                                                             │
│                                                                             │
│ This refactoring extracts breach sources into a plugin architecture under   │
│ core/osint/sources/, adds a BreachManager orchestrator, implements two new  │
│ API integrations (Snusbase, DeHashed API), and fills in placeholder methods │
│  (GitHub leaks, paste monitoring). The existing check_data_breaches() and   │
│ check_vulnerability() return formats remain unchanged for backward          │
│ compatibility.                                                              │
│                                                                             │
│ Files to Create                                                             │
│                                                                             │
│ 1. core/osint/sources/base.py -- Abstract Base Class                        │
│                                                                             │
│ - SourceType enum: API_KEYED, API_FREE, LOCAL_DB, WEB_SCRAPE                │
│ - SourceHealth dataclass: available, last_error, response_time_ms           │
│ - BreachResult dataclass: HIBP-compatible fields (Name, Title, BreachDate,  │
│ Description, DataClasses, IsVerified, IsSensitive, Source) with to_dict()   │
│ method                                                                      │
│ - BreachSource ABC with:                                                    │
│   - Properties: name (str), source_type (SourceType), rate_limit_delay      │
│ (float, default 1.0)                                                        │
│   - Methods: query(email) -> List[BreachResult], health_check() ->          │
│ SourceHealth, is_configured() -> bool                                       │
│   - Contract: query() never raises -- catches exceptions internally,        │
│ returns []                                                                  │
│                                                                             │
│ 2. core/osint/sources/manager.py -- BreachManager                           │
│                                                                             │
│ - __init__(config=None): reads config.osint.breach_sources for              │
│ enabled/priority settings                                                   │
│ - register_source(source) / unregister_source(name)                         │
│ - get_enabled_sources() -> List[BreachSource]: filters by enabled and       │
│ is_configured(), sorts by priority                                          │
│ - query_all(email) -> Dict: queries each enabled source in priority order,  │
│ deduplicates by (Name, Source), returns:                                    │
│ {'email', 'pwned', 'breach_count', 'breaches', 'sources_queried',           │
│ 'sources_failed', 'query_time_ms'}                                          │
│ - health_check_all() -> Dict[str, Dict]: runs health_check() on all         │
│ registered sources                                                          │
│ - _log_source_query(source_name, email, result_count, success): per-source  │
│ audit logging to data/osint_logs/breach_queries_YYYY-MM.jsonl (email hashed │
│  with SHA256 before logging)                                                │
│ - Uses RateLimiter from utils/rate_limiter.py (reuse, don't rebuild)        │
│                                                                             │
